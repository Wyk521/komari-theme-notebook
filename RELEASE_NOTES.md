# Komari Notebook v0.8.4

本版本修复刷新页面时短暂显示固定三列巨型空白骨架卡的问题。

## 修复内容

- 在样式表加载前恢复上次确认的三列或四列设置
- 初次访问没有布局缓存时直接使用主题默认的四列布局
- 将固定 `500px` 的空白板改为紧凑、结构化的作业本卡片骨架
- 三列仅显示三个占位，四列显示四个占位；平板和手机分别显示两个、一个
- 同步缓存布局与其他主题外观设置，减少刷新阶段的视觉跳变

## 验证

- 慢接口状态下，三列缓存首帧直接显示三列骨架
- 慢接口状态下，四列缓存首帧直接显示四列骨架
- 无缓存首访使用默认四列，不再出现固定三列
- 移动端只显示一个紧凑占位且没有横向溢出

## 安装

下载 Release 附件 `KomariNotebook-v0.8.4-install.zip`，在 Komari 管理后台的主题管理页面上传并启用。
