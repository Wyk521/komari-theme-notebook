# Komari Notebook v0.8.6

本版本修复节点卡片与详情入口的悬停抖动，并把节点详情重新收拢到作业本纸张语言中。

## 修复内容

- 卡片悬停时不再改变垂直位置，避免鼠标位于边缘时反复触发移入与移出
- 保留稳定的边框和阴影反馈，不影响卡片点击与详情入口
- 节点详情顶部采用三张轻薄的横线纸分栏，以红色页边线、横格和手写体标题恢复作业本辨识度
- 数据行取消独立描边，不再出现密集的框套框；只依靠纸张横线完成阅读对齐
- 图表和 Ping 区只保留一次纸张分组，内部信息改用细墨线与页边标记区分
- 标题分隔和图表网格改为极淡连续线，形成统一且克制的纸面层级
- 样式文件名按内容自动生成哈希，避免主题覆盖安装后继续读取旧浏览器缓存
- 主题设置新增默认关闭的 7/30/90 天长期历史开关，同时控制负载与 Ping 延迟范围
- 修复卡片详情按钮悬停时的位移抖动

## 安装

下载附件 `KomariNotebook-v0.8.6-install.zip`，在 Komari 管理后台的主题管理页面上传并启用。
